# 🗺️ CYT PDF 工具：後續優化方向

本文件紀錄了專案的待辦清單與未來可能的開發計畫。

## 🛠️ 近期任務 (核心功能完善)
- [ ] **PDF 轉 JPG 介面實作**：將 `pdf_utils.py` 的邏輯整合進 `ConvertPage`。
- [ ] **Poppler 自動偵測**：若系統未安裝 Poppler，自動提示下載連結或在設定頁面手動指定路徑。
- [ ] **拖放功能增強**：整合 `tkinterdnd2` 以實現真正的檔案拖曳進介面，而不僅是點擊開啟。

---

## 🚀 未來藍圖 (進階功能)
- [ ] **PDF 壓縮功能**：透過優化圖片品質與移除冗餘物件，降低 PDF 體積。
- [ ] **浮水印預覽**：在介面上即時預覽浮水印的位置與透明度效果。
- [ ] **光學字元辨識 (OCR)**：整合 Tesseract OCR，將掃描版 PDF 轉為可搜尋的文字版。
